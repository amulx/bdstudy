3、查找和替换
	通过sed来查找
		:s/i/-/g
		:1,10s/i/-/g   第一行到第十行
		:1,$s/i/-/g

4、读取和保存
	:r file 将另外一个file的内容读入当前文件
	1,20 w file 将1-20行的内容写入到一个新的file中
	1,$ w file 将整个文件写入一个新的文件
		
		set number
		:1,3 w >> amu.txt
5、vim的设置
	:set 显示常用的配置
	:set all 显示所有的配置选项

6、撤销操作
	u  撤销最近一次的修改
	ctrl-r  取消最后一次的撤销  redo
	U 撤销所有修改

7、粘贴操作
	yy  复制当前行
	yl  复制当前字母leter
	yw  复制当前单词 word
	p粘贴在行上/左侧  P粘贴在行下/右侧

8、命令模式下的一些技巧
	dtc 删除从光标到c之间的所有字符
	rc 将光标下的字符替换成c
	dw 删除游标处所在的字符
	5dd 删除5行数据
	5yy 复制5行
	5x	删除5个字符	
	R    进入替换状态 esc退出
